Operating Systems
Created and published many repositories using Windows and Linux. The flavors of Linux used for development are Mint Cinamon 20.2, Ubuntu 20.04 LTS, and Qubes 4.0.4. Linux Mint Cinamon is a easy to use operating system that allows users to navigate with ease. Linux Mint Cinamon is stuningly beautiful and smooth. Linux Mint also allows for robust security and productivity features like virtual desktops. Ubuntu is an open-source operating system that allows users to perform a large variety of tasks with pre-installed apps. Ubuntu is also secure with firewalls and built-in virus protection. Ubuntu also supports a large variety of hardware. Qubes OS is a reasonably secure operating system relayed over the Tor Network. The Tor Network relays requests in 3 layers ensuring moderate anonymity. Qubes OS also allows users to store applications in Virtual Machines (VM) for robust security, while monitoring their activity.
